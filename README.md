# gateway-service

Simple Spring Boot microservice (Gradle) configured for PostgreSQL and basic security.

- Spring Boot: 3.2.2
- io.spring.dependency-management: 1.1.4
- Java: 17

## Notes
- Fill datasource properties in `src/main/resources/application.yml`.
- This project does **not** include Gradle wrapper files. Run with your local Gradle or generate wrapper using `gradle wrapper`.
